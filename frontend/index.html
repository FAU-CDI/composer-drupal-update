Getting Started

    
    
      
  1. Load your composer.json by dropping it onto the drop zone, clicking it to browse, or pasting it into the textarea and clicking Done Editing.
    2. 
      
  2. Switch to the Packages tab. You'll see all your Drupal modules and Composer packages with their current versions. Use the dropdowns to pick new versions.
    3. 
      
  3. Click Apply to write the selected versions back into the JSON. The tab title shows (*) when you have unsaved changes.
    4. 
      
  4. Back on the composer.json tab, click Download to save the updated file.
    5. 
    

    
Tabs

    

      
composer.json

      
View, edit, upload, and download your composer.json. Click Edit to make manual changes; click Done Editing to re-parse.

      
Packages

      
Lists Drupal and Composer packages separately. Each row links to the project page. Pick a version from the dropdown and click Apply.

      
Commands

      
Shows ready-to-run composer require "...version" --no-update commands for every package in your require section, followed by a single composer update --dry-run. Copy them into your terminal to apply the same updates programmatically without replacing the file.
